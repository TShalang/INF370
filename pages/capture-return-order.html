Capture Return Order

The purpose of this screen is to show all the customer return the system has on record. The admin can capture a new customer return by the selecting the add new customer return link that leads to the add customer return page, on this page the admin needs to input on all the textboxes and select create. Alternatively, the admin can select cancel and redirected back to the customer return page. The admin cannot update and delete a customer return due to business rules
