Generate Quotation

The purpose of this screen is for the admin to generate a quotation after viewing a quotation request. The screen displays the customers details and the items they want to be quoted on. The admin can view the design details that will have the product type, image to print (if selected), job type and what positions. The admin will enter the price per quantity item and select to generate the quotation.
